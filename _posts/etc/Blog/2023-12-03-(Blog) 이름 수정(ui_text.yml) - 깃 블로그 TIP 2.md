# Intro

**`_data/ui-text.yml` 을 수정하여 지정된 이름을 수정해보자!**

* 예 : "최근 포스트" -> "Recent Posts"
* 참고로 스타일링은 따로 한겁니다!

# 이름 수정 - ui_text.yml

**TOC라벨, 최근 포스트** 이름을 각각 **"목차", "Recent Posts"**로 변경

* `_data/ui-text.yml` 에서 아래와 같이 수정
  * `toc_label : "목차"`
  * `recent_posts  : "Recent Posts"`
* 참고로 Korean 언어 부분에서 수정!
